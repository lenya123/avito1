# Навигация приложения

## Сайт клиента (5 разделов в BottomNav)

| Раздел | URL | Описание |
|--------|-----|----------|
| **Каталог** | `/catalog` | Товары с фильтрами, избранное, уведомления о наличии |
| **Заказы** | `/stats` | Список заказов + статистика/графики (premium) |
| **Обучение** | `/education` | Гайды: выбор товара, оформление, возвраты, продажи, FAQ |
| **Помощь** | `/support` | AI-чат (premium) + FAQ для всех |
| **Профиль** | `/profile` | Баланс, подписка, рефералы, лидерборд, Avito API, экспорт |

### Дополнительные страницы клиента

- `/order/[productId]` — оформление заказа (из каталога)
- `/stats/[orderId]` — детали заказа (из списка заказов)
- `/profile/subscription` — управление подпиской
- `/profile/notifications` — настройки уведомлений

## PWA отправщика (5 разделов в BottomNav)

| Раздел | URL | Описание |
|--------|-----|----------|
| **Главная** | `/shipper` | Сводка дня |
| **Отправка** | `/shipper/ship` | Заказы к отправке |
| **Возвраты** | `/shipper/returns` | Прибывшие возвраты |
| **Статистика** | `/shipper/stats` | Показатели работы |
| **Настройки** | `/shipper/settings` | Профиль отправщика |

| Раздел | URL | Описание |

| **Dashboard** | `/owner/dashboard` | Метрики, графики, алерты |
