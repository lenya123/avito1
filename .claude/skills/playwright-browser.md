# Playwright Browser (MCP)

Управление реальным Chromium-браузером через Playwright MCP. Подключён глобально (`~/.claude.json`).

## Изолированные сессии

Каждый чат получает свою изолированную сессию браузера:

- Параллельные чаты не конфликтуют
- Авторизации копируются из `~/.playwright-session/`
- После завершения чата сессия удаляется автоматически
- Изменения авторизаций внутри чата **не сохраняются** в основную сессию

## Когда использовать

**Используй Playwright когда:**

- Нужна интерактивность (клики, ввод, навигация)
- Требуется авторизация (Supabase, Avito, Telegram Web)
- Страница — SPA (WebFetch вернёт пустой HTML)
- Нужна визуальная проверка проекта (`localhost:3000`) — перед навигацией проверь `lsof -i :3000` и не запускай `npm run dev` если сервер уже работает
- WebFetch не справился (динамический контент, JS-рендеринг)

**Используй WebFetch/WebSearch когда:**

- Простая статическая страница без JS
- Поиск общей информации в интернете
- Документация на обычном HTML

## Рабочий процесс

**Стратегия: snapshot → analyze → action → verify**

1. **Snapshot первым делом** (`browser_snapshot`)
   - Accessibility tree — основной инструмент навигации
   - Быстрее и дешевле screenshot
   - Даёт `ref` для кликов/заполнения форм

2. **Screenshot только когда нужно**
   - Визуальная проверка UI (графики, дизайн)
   - Диагностика layout-багов
   - Когда snapshot не даёт достаточно информации

3. **Действия через `ref`**
   - Используй `ref` из snapshot для точных кликов/заполнения
   - После действия — новый snapshot для проверки результата

4. **Verify результат**
   - Новый snapshot после действия
   - Console/network при ошибках (`browser_console_messages`, `browser_network_requests`)

## Авторизации

**Для повторного использования между чатами:**

1. Залогинься в браузере (Supabase/Avito/Telegram Web/etc)
2. Cookies/localStorage автоматически сохраняются в `~/.playwright-session/`
3. Все будущие чаты получат копию этой авторизации
4. **Важно:** изменения внутри чата не влияют на главную сессию

## Скриншоты

**Во время работы:**

- Playwright сохраняет в `.playwright-mcp/`

**После завершения задачи:**

```bash

```
